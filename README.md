# Call Center Analysis

## Project Overview
This project provides a comprehensive analysis of call center performance metrics, including agent productivity, customer satisfaction, and call trends. The analysis leverages data-driven insights to identify top performers, areas for improvement, and actionable recommendations.

## Data Source
- **Source**: Kaggle
- **Dataset**: Call Center Performance Data
- **Original File**: `excel_callcenter.xlsx` (Raw Data folder)

## Tools & Technologies Used
- **Data Cleaning**: Power Query Editor
- **Analysis & Visualization**: Microsoft Excel
- **Dashboard**: Excel Dashboard with visualizations

## Project Structure

## Key Findings & Analysis

### 1. Executive Summary
- **Total Calls Handled**: 5,000
- **Successfully Resolved**: 3,646 calls (73% resolution rate)
- **Team Size**: 8 agents

### 2. Agent Productivity Metrics
- **Top Answerer**: Jim (536 calls answered)
- **Most Resolved**: Jim (483 resolved calls), followed by Becky (463)
- **High Efficiency**: Martha achieved strong completion rate with 461/514 calls resolved
- **Workload Distribution**: Relatively even distribution across the team with some performance variations

### 3. Customer Satisfaction (CSAT)
- **Average CSAT Rating**: 3.3 - 3.4 out of 5
- **Peak Performers**: Dan and Martha (3.4 rating)
- **Performance Gap**: Joe (3.2 rating) - identified for targeted coaching

### 4. Call Topics & Volume Analysis
Four main call categories identified:
- Admin Support
- Contract-related Issues
- Payment-related Issues
- Streaming-related Issues

**Key Insight**: Streaming and Payment-related issues generate the highest call volume, indicating primary customer pain points.
